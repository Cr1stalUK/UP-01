Разработка web-сайта для УП.01

Целью разработки сайта является представление проделанной работы на практике. Контент связан с разработкой программных модулей, созданием АИС, 1С программированием, разработкой мобильного приложения и также с созданием сайта.

Среда разработки для создания сайта – Visual Studio Code

Использовались: язык разметки HTML (для создания структуры и содержимого) и язык стиля CSS (для описания представления и макета)
