Разработка мобильного приложения

Приложение «AutoMat» предназначено для использования его в целях записи на ремонт или проверку своего автомобиля в автосервисе. Интерфейс приложения «AutoMat» достаточно прост и интуитивно понятен.
Платформа: Android
Среда разработки: Android Studio
Язык программирования: Java
Используемые графические элементы: TextView, EditText, ImageView, Button, ListView, FragmentContainerView
